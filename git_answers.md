1. Hvað gera eftirfarandi Linux skipanir?

    * cd - answ: Breytir um moppur sem unnid er i.
    * ls - ans: Synir allar skrar og moppur i nuverandi moppu.
    * pwd - ans: Birtir nuverandi vinnusvaedi.
    * mkdir - ans: Byr til nyja moppu.

2. Hvað gera eftirfarandi Git skipanir?

    * git clone - ans: Byr til afrit af branchi a tolvu notendans.
    * git status - ans: Byrtir stodu nuverandi vinnusvaedi.
    * git diff - ans: Byrtir breytingarnar sem thu hefur gert.
